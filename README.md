# The Witness Stand [![Build Status](https://app.travis-ci.com/mrrusof/the-witness-stand.svg?branch=master)](https://travis-ci.com/github/mrrusof/the-witness-stand)

## Requirements

- Docker 18.03.0-ce
- Bash 4.3
- Make 4.1
- Bats 0.4.0
- jq 1.5-1-a5b5cbe
- Ruby 2.4.0

## Make Targets

```
make [all|build|test|clean]
```
